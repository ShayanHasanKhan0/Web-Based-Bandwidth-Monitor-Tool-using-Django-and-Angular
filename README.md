# Web-Based-Bandwidth-Monitor-Tool-using-Django-and-Angular
Web-Based Bandwidth Monitor Tool Using Django and Angular


# For Angular
run these commands
+ npm i
+ ng serve

# For Django
+ python3 -m pip install Django
+ python3 pip install snmp
+ python3 pip install netifaces
+ python3 pip install speedtest-cli
+ python3 manage.py runserver
